# 30 Days Of Python Exercise Solutions

This repository includes all the solutions for the exercises from the 30 Days Of Python Repository accordingly.

The file names are named according to the day exercises.

Some folders are present as a result the solutions to some of the exercises. The files in the data folder contains some of the file requirements to solve some of the exercises. Mostly those that expects you to ectract data from a file.

## Note:

- Days that does not have any solutions in it mean there were no exercises given. e.g Day 15, 23, 24.
- Most if not all files have a lot of commented out code blocks (especially ones with the "print" function). This is to enable you to choose which code to run and print by just uncommenting the code block.
